Цель: Создания приложения для передачи данных между клиентом и сервером через socket, а также вывод значений на клиенте и сохранения всех данных на сервере

Задачи: 
1) Передача данных на клиента с последующим показом графиков по времени
2) Сохранение данных на сервере для более точного анализа по графикам

Технологии: 

Интерфейс: Tkinter
Основные библиотеки: matpotlib, socket

Входные данные: Данные с 5 датчиков(температура 1, температура 2, температура 3, температура 4, Напряжение)

Выходные данные: Текстовый файл с сохраненными значениями в форме время-значение.
